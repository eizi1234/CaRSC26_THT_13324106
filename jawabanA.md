  ##### a. beberapa command di git adalah : 
    ###### - git init = meginisialisasi pembuatan repositori baru
      ###### contoh : git init
    ###### - git clone = menginisialisasi duplikasi repositori
      ###### contoh : git clone (link) _file_
    ###### - git add = menambahkan sesuatu (contoh: file) pada suatu file
      ###### contoh : git add (file)

      ###### contoh : git branch (jika ingin menambahkan branch)
      ######        : git branch namagw (jika ingin menambahkan branch dengan nama)
      ######        : git branch -b namagw (jika ingin menambahkan branch dengan nama lalu memindahkannya ke branch baru)
